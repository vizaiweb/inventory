📌 掃描流程
1. 點擊「啟動掃描器」並允許鏡頭權限
2. 將鏡頭對準條碼/QR Code
3. 掃描成功後鏡頭會自動停止
4. 系統會自動查詢資料庫：
  • ✅ 有相同條碼 → 開啟該貨品資料
  • 🆕 無相同條碼 → 開啟新增表單
5. 填寫/確認貨品資訊後點擊「儲存」
6. 點擊「掃描下一筆」繼續掃描其他貨品

📌 編輯貨品
在「貨品清單」點擊貨品即可編輯

📌 資料備份
可匯出 JSON 檔案備份，或匯入還原資料
